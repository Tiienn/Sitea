# Feature specs

Created by /spec, implemented by /build, verified by /review.

Lifecycle: `ready-to-build` → `built` → `approved` | `changes-requested`

- /spec <idea> — interviews you until the feature is fully understood, writes the spec here
- /build [name] — implements the spec (defaults to the only ready-to-build spec)
- /review [name] — adversarial review against the spec (defaults to the only built spec)
